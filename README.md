# Steps to run backend server

```sh
$ npm install
```
```sh
$ npm start
```

This will run a backend server on localhost:5000 with message succesfully connected to DB.

# Steps to run client server

```sh
$ cd client
$ npm install
```
```sh
$ npm start
```

This will run a client server on localhost:3000.
